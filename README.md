# texvcTokens
[![Build Status][1]][2]

Prints the texvvc tokens idetified by texvcjs.

## Installation

Node version 0.10 are tested to work.

Install the node package dependencies with:
```
npm install
```
Ensure everything works:
```
npm test
```
